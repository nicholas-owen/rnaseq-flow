rnaseq-flow — Pipeline Overview

Directed acyclic graph of the modular bulk RNA-seq pipeline. Boxes are Nextflow processes; rounded stores are data inputs/references. Solid arrows are required data flow; dashed arrows are optional inputs or stages that run only when the relevant flag is set.

Workflow map — metro / underground style

The same RNASEQ workflow drawn as an nf-core-style transit map. Each coloured line is one aligner route; white pills are interchange steps shared by routes; rings are single-route stops. DESeq2 and edgeR are two parallel DE routes off the gene counts — both run, and only the DESeq2 route carries on to GSEA and gProfiler.

Conditional execution

Which nodes run depends on the aligner, optional flags and the --stop_at staging parameter.
GateEffect on the graph
--alignerOne of STAR / HISAT2 / Salmon / Kallisto runs. DESeq2 + edgeR + enrichment run for every aligner — STAR/HISAT2 feed them via featureCounts, Salmon/Kallisto via tximport (needs --gtf). rMATS and BigWig are STAR/HISAT2 only.
--stop_at preQCStops after FastQC + fastp.
--stop_at postQCStops after alignment, SAMtools index, RSeQC and BigWig.
--stop_at DEStops after gene quantification, DESeq2 and edgeR.
--ctat_libEnables STAR-Fusion (STAR only).
--isoform_switch + --transcript_fastaEnables IsoformSwitchAnalyzeR (Salmon only).
--dtuEnables the DEXSeq differential transcript usage test (Salmon or Kallisto; needs --gtf).
--diffspliceEnables the edgeR diffSpliceDGE test — exon-level usage on STAR/HISAT2, transcript-level usage on Salmon/Kallisto (needs --gtf).
--gmtEnables GSEA. gProfiler runs whenever differential expression runs.
MultiQC and the Quarto report always run and aggregate whatever stages executed.
